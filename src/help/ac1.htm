Hinweise zur Emulation des AC1

Emulierte Hardware

  • 4 kByte ROM für Monitorprogramm und Mini-BASIC
  • 60 kByte RAM inkl. 2 kByte Bildwiederholspeicher
  • 64x32 Zeichen Bildschirmgröße mit erweitertem Zeichengenerator
  • Kassettenrecorderanschluss


Im ROM enthaltene Software

  • Monitorprogramm Version 3.1
  • Mini-BASIC Version 2.1
