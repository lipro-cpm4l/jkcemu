RAM-Floppies

Einige der emulierten Systeme verfügen im Original über eine oder zwei RAM-Floppies. Diese werden auch emuliert.

JKCEMU bietet Ihnen die Möglichkeit, auf die in den emulierten RAM-Floppies gespeicherten Daten direkt zuzugreifen, d.h. diese Daten können als Abbilddatei gespeichert und auch wieder geladen werden. Rufen Sie dazu im Emulatorfenster im Menü Datei den Menüunkt RAM-Floppies... auf.

In den Einstellungen zu den einzelnen emulierten Systemen, bei denen RAM-Floppies emuliert werden, können Sie auch Abbilddateien angeben, die automatisch geladen werden. Dies ist eine besonders bequeme Möglichkeit, zu Beginn der Arbeit die RAM-Floppies mit sinnvollen Daten vorbelegen zu lassen.
