Weitere Funktionen

Hier finden Sie eine Beschreibung der weiteren Funktionen, die Ihnen der Emulator bietet.

1. Weitere Funktionen im Menü Datei

1.1. Schnappschuss der Bildschirmausgabe

Mit den Funktionen in diesem Untermenü können Sie die gerade angezeigte Bildschirmausgabe als Bild oder Text extrahieren und in eine Datei speichern, in die Zwischenablage kopieren oder im Bildbetrachter bzw. Texteditor öffnen.

2. Weitere Funktionen im Menü Bearbeiten

2.1. Kopieren

Sie können in der Bildschirmausgabe Text markieren und diesen in die Zwischenablage kopieren. Bei einigen Systemen, u.a. bei denen mit einer 7-Segment-Anzeige, steht die Funktion allerdings nicht zur Verfügung.

2.2. Einfügen

Mit dieser Funktion fügen Sie Text aus der Zwischenablage in die Bildschirmausgabe ein, aber nur scheinbar, denn in Wirklichkeit wird dieser Text an das im Emulator laufende Programm übergeben. Dabei wird für jedes einzelne einzufügende Zeichen das Betätigen der jeweiligen Taste simuliert. Das dauert prinzpbedingt relative lange. Bei einigen emulierten Systemen gibt es auch die Möglichkeit, das Einfügen durch Einschalten einer speziellen Option zu beschleunigen.

Durch das Prinzip der Simulation von Tastaturbetätigungen kann es auch vorkommen, dass sich bestimmte Zeichen nicht einfügen lassen. Das ist genau dann der Fall, wenn das emulierte System dafür keine Taste bzw. Tastenkombination kennt. In dem Fall erscheint ein entsprechender Hinweis, und das Einfügen wird angehalten. Sie können mit Hilfe des Menüpunktes Einfügen fortsetzen später wieder fortsetzen.

3. Weitere Funktionen im Menü Extra

3.1. Schachbrett

Wird gerade ein Schachcomputer oder Lerncomputer mit integriertem Schachprogramm emuliert, können Sie hiermit ein Fenster zur Anzeige der Schachbrettbelegung öffnen.

3.2. Pause

Mit dieser Funktion halten Sie die Programmausführung des Emulators an. Mit einem nochmaligen Klick auf den Menüpunkt setzen Sie die Programmausführung wieder fort.

3.3. Nicht maskierbarer Interrupt (NMI)

Mit dieser Funktion lösen Sie einen nicht maskierbaren Interrupt aus. Dazu wird eine Interrupt-Routine gestartet, die auf Adresse 0066h installiert sein muss.

Achtung! Wenn sie einen NMI auslösen, ohne dass eine entsprechende Interrupt-Routine installiert ist, kann das zum Absturz des im Emulator laufenden Programms und damit zu Datenverlust führen.

3.4. Zurücksetzen (RESET)

Mit dieser Funktion setzen Sie den Emulator zurück. Dabei wird das Auslösen eines RESET-Signals emuliert. Das emulierte System startet neu und meldet sich mit der Eingabeaufforderung.

3.5. Einschalten (Power On)

Der Emulator löscht den Arbeitsspeicher und führt einen RESET aus.
